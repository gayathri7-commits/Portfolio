## 🌟 Visual Preview & Sections

The portfolio is architected as a sleek, low-fatigue dark-themed landing page structured into high-impact informational blocks:
* **Hero Section:** Introduction featuring an active status tag (`../AVAILABLE FOR OPPORTUNITIES`), professional profile picture, and a high-level metrics dashboard showing academic performance, internship experience counts, and graduation year.
* **Skills Matrix:** A curated display of core engineering capabilities spanning languages, frameworks, and database environments.
* **Projects Showcase:** Highlighted repository integrations with active deployment links.
* **Experience & Achievements:** Detailed chronologies of internships and organizational milestones.
* **Call to Action (CTA):** Accessible links to download my professional Resume, connect on LinkedIn, or send an Email directly.

## 🛠️ Tech Stack & Implementation Details

* **HTML5:** Structured semantic markup designed for accessibility and search engine visibility.
* **CSS3:** Custom styles leveraging an elegant palette (Deep Obsidian `#121212`, Coral Accent, and Stark White), modern layout grids, flexbox alignments, and smooth button hover micro-interactions.
* **JavaScript (ES6):** Controls smooth-scroll page navigation, interactive elements, and responsive action toggles.
